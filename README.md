# 힐링송 프로젝트 - 청춘의 포말

두 곡의 힐링송을 소개하는 웹페이지입니다. 썸네일 이미지와 간단한 소개, 유튜브 재생 링크를 제공합니다.

## 팀원
- 김나림
- 유현정
- 정선아

## 역할 분담
- 김나림: (예: 헤더/타이틀 구성)
- 유현정: (예: 노래 섹션 레이아웃 및 내용 정리)
- 정선아: (예: footer 제작자 정보, 링크 정리, README 작성)

## 파일 구성
- `index.html` : 페이지 구조(시맨틱 태그 사용)
- `style.css` : 디자인 스타일
- `script.js` : 간단한 동작(링크 클릭 로그, footer 연도 표시)
- `청춘의 포말.jpg` : 썸네일 이미지

## 실행 방법
1. 이 저장소를 다운로드 또는 클론합니다.
2. `index.html` 파일을 브라우저로 열면 웹페이지가 실행됩니다.
